## 使用技術

- 後端：Python、Django、Alpine.js
- 部署：AWS EC2
- 套件：
  - django-tailwind
  - django-browser-reload
  - django-qr-code
  - beautifulsoup4

### 執行環境

1. poetry shell 虛擬環境
2. poetry install 下載 python 相應套件
